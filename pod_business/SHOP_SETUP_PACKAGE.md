## QUICK UPLOAD COMMANDS

**Convert SVG to PNG (PowerShell):**
```powershell
# Install Inkscape first: https://inkscape.org/
$svgFiles = Get-ChildItem "C:\Users\quent\.openclaw\workspace\pod_business\designs\*.svg"
foreach ($file in $svgFiles) {
    $output = $file.FullName -replace '\.svg$', '.png'
    inkscape $file.FullName --export-filename=$output --export-width=7632 --export-height=6480
}
```

**Or use online converter:** https://cloudconvert.com/svg-to-png
