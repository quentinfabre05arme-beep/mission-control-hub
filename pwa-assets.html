Setup Instructions

To set up your PWA:

  1. Download the icons above and save them to an icons/ folder
  2. Download the splash screens and save them to a splash/ folder
  3. Place manifest.json and sw.js in your root directory
  4. Serve the files over HTTPS (required for PWA features)
  5. Open the app in Chrome/Edge/Safari and look for the install prompt
