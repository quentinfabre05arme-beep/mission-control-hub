## Key Decisions & Learnings

1. **Manual model switching preferred**: User explicitly chose manual over automatic model routing
2. **Vercel static deployment**: Direct HTML file updates propagate immediately after git push
3. **Browser automation reality**: OpenClaw CLI evaluate/type commands have parsing bugs; Playwright Python works as alternative
4. **Rate limiting**: Twelve Data free tier very restrictive (8 calls/minute); quote endpoint used to minimize calls
5. **PowerShell encoding**: Emoji paths break scripts; ASCII references used for automation while emojis remain for display
6. **Deployment gap detection**: Static files can miss Vercel deployment despite git push — periodic URL audits needed
