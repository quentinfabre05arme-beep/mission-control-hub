Open `command_center.html` for the latest unified experience, or `index.html` for v5.3 with Competitor Tracking.

## Command Center Features (v7.0)

### Command Palette (⌘K)
- **Quick Actions**: Create post/thread, view reports
- **Navigation**: Jump to any section
- **External Links**: Access legacy dashboards

### Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| ⌘K | Open command palette |
| ⌘N | Create new post |
| ⌘T | Create new thread |
| Esc | Close palette/dialog |

### Layout
- **Hero Stats**: 4-column live-updating metrics
- **Bento Grid**: Modern card-based layout
- **Progress Tracker**: Visual growth mission progress
- **Content Pills**: Interactive content pillar cards
- **Activity Feed**: Recent actions with timestamps
- **AI Insights**: Real-time recommendations
- **Quick Actions**: 4-button action grid
- **Charts**: Engagement trends & follower growth

### Live Features
- Real-time clock (Europe/Paris timezone)
- Simulated follower count updates
- Toast notifications for actions
